# NYP GLOBAL IMPACT HUB

Proyecto web desarrollado con React, Node.js, Express y Docker.

## Tecnologías utilizadas

- React
- Vite
- Node.js
- Express
- Docker
- Docker Compose

---

# Clonar repositorio

```bash
git clone https://github.com/eidymanzo/Pagina_Web_NYP_GLOBAL_IMPACT_HUB.git
```

# Entrar al proyecto

```bash
cd PAGINA_WEB_NYP_GLOBAL_IMPACT_HUB
```

# Ejecutar aplicación

```bash
docker compose up --build
```
